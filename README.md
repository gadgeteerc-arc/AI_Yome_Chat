# AI嫁チャ (AI Yome Chat)

モダンでプレミアムなUIを備えた、ローカルAIエージェント向けのチャットインターフェースだお！(( ・∀・)ｲｲ!!)

## 特徴

- **Frost & Noise ガラスモーフィズム**: 透明感とノイズテクスチャを組み合わせた、2026年トレンドの超美麗UI。
- **Bento Grid レイアウト**: 整理された機能的なデザイン。
- **カスタマイズ性**: エージェント名、ショートカットキー、アクセントカラー、透過度、フォントサイズなどがUIから即座に変更可能。
- **マルチデバイス対応**: レスポンシブ設計で、どんな画面サイズでも美しく表示。
- **思い出ギャラリー**: 生成された画像を一覧表示し、背景に設定可能。

## ディレクトリ構成

- `/frontend`: React + Vite によるフロントエンドソース。
- `/backend`: Node.js + Express によるバックエンド（API/WebSocket）ソース。
- `/data`: チャット履歴 (`messages.json`) や立ち絵・画像の保存場所。

## 前提条件

このアプリを動かすには、**Node.js** (v18以上を推奨) がインストールされている必要があるお！

- [Node.js 公式サイト](https://nodejs.org/) からダウンロードしてインストールしてね。

## セットアップ手順

### 1. リポジトリをクローン
```bash
git clone https://github.com/gadgeteerc-arc/AI_Yome_Chat.git
cd AI_Yome_Chat
```

### 2. バックエンドの起動
```bash
cd backend
npm install
npm start
```
※ デフォルトで `http://localhost:3001` で起動するお。

### 3. フロントエンドの起動
別のターミナルを開いて：
```bash
cd frontend
npm install
npm run dev
```
※ `http://localhost:5173` （または表示されたURL）にブラウザでアクセスしてね！

## 使い方

1. 画面右上の **ギアアイコン（設定）** をクリック。
2. 立ち絵フォルダ (`data/expressions`) や画像保存フォルダ (`data/gallery`) のパスを確認・設定して保存。
3. エージェント（AI等）側から `data/messages.json` に書き込めば、リアルタイムでチャットが始まるお！
4. **裏技**: ヘッダーのタイトル部分をダブルクリックすると……？

## エージェント連携 (Antigravity等)

このプロジェクトには、Antigravity等のAIエージェントと連携するための設定ファイルが同梱されているお！

1. `.agent/` フォルダを、あなたのエージェントのワークスペース直下にコピー。
2. エージェントに対して「YomeChatスキルを使ってチャットして」と指示。
3. これでAIが自律的に `messages.json` を監視・返信し、UIに反映されるようになるお！( ・∀・)ｲｲ!!
